# Waitlist Registration

A minimal, clean waitlist registration page.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

## Features

- Clean, minimal design
- Email validation
- Duplicate prevention
- SQLite database storage
- Error handling

## Database

Emails are stored in `waitlist.db` SQLite database. The database is created automatically on first run.
